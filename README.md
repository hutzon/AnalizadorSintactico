# AnalizadorSintactico

Tarea de la clase de Compiladores. Incluye los ejercicios originales en Java
generados con ANTLR y una versión adicional escrita a mano en Python.

## Contenido

- `Ejercicio 1/` y `Ejercicio2/`: proyectos Java con gramáticas ANTLR para
  practicar análisis sintáctico.
- `analizador_python/`: implementación manual en Python de un analizador léxico
  y sintáctico que valida y evalúa expresiones aritméticas.

Consulta los README de cada carpeta para instrucciones de compilación y uso.
